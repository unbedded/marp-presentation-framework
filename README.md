## Quick Start

```bash
# Install Marp CLI
npm install -g @marp-team/marp-cli

# Edit slides
vim source/presentation.md

# Build presentation (HTML + PDF)
make marp

# Present
firefox artifacts/presentation.html  # Press F11 for fullscreen
```
